Install SQL Server 2000

Launch SQL Server Enterprise Manager
- Open "SQL Server Properties (Configure)" window
- Under "Security" tab, enable authentication for "SQL Server and Windows"
- Create a new database "cdb_cfs"
- Create a new login for user "postgres" 
  SQL Server Password "p0stgres"
  Role "System Administrator"
  Database Access: permit "cdb_cfs"
